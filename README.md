# Site Web "Les Bulles de Joie" - Résultats Scolaires

Site web moderne et responsive pour afficher les résultats scolaires de l'école "Les Bulles de Joie" à Parakou, Bénin.

## Fonctionnalités

- 🌍 **Multilingue** : Français, Anglais, Espagnol
- 📱 **Responsive Design** : Optimisé pour tous les appareils
- 🎨 **Interface moderne** : Design glassmorphism avec animations
- 🔐 **Sécurité** : Authentification par mot de passe
- 📊 **Statistiques détaillées** : Moyennes, meilleures notes, etc.
- 🖨️ **Impression** : Bulletins imprimables
- 🎉 **Effets visuels** : Confettis, animations 3D
- 🏫 **Multi-niveaux** : CI, CP, CE1, CE2
